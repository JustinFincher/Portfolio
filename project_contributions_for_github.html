Contributions For GitHub

An app with 2D&3D Widgets / Watch Complications to show your GitHub contributions graph on iOS. (Featured on Product Hunt as 4th on 2016.11.22!)

App Store GitHub Product Hunt

"Contributions For GitHub" reads your public (also private if you enable the "show private" contribution settings on your GitHub profile) commits from its svg source and shows the data by 2D / 3D using UIImage / SceneKit.
