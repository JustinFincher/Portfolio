"Skyline LWP", my side project in Dec 2017. It offers some functionalities similar to the Pixel Live Wallpaper by B-Reel, but it's customizable in location, zoom, rotation, etc.
Don't forget to check out the Trailer on YouTube. :-)

  • App built using Unity and Mapbox SDK.
  • Landing page is made using three.js.
  • Promo video #1 is made using 3dsMax/Vray/C4D/Octane Render, while trailer(#2) using AE/E3D.
